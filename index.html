Percentage Allocator

Distribute values across line items, always summing to 100%

How to use

  • Total amount — set what you're dividing (e.g. $1,000). Each item shows its calculated share.
  • Drag slider — drag the colored handle to adjust; other unlocked items auto-redistribute.
  • Click % or amount — click either number to type an exact value.
  • Click name — click an item name to rename it.
  • Lock — click the lock icon to freeze an item's percentage; other unlocked items redistribute around it.
  • Link groups (A–E) — items in the same group scale proportionally when any one is dragged.
